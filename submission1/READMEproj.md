# CppGame Project

## How to Run the Files

1. **Compile the Code**:
    Compile with:
    ```bash
    g++ sub1.cpp Objects.cpp -o run
    ```

2. **Run the Program**:  
    Run with:
    ```bash
    ./run
    ```

## How to Input Data

- give the program a type and size, ex, `512, 4` or `256, 3`
- type `up`, `down`, `left`, `right` to slide tiles in that direction
- `Ctrl-Z` or `exit` to quit
